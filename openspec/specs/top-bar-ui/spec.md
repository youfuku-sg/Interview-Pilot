## MODIFIED Requirements

### Requirement: Top bar button set
トップバー（フローティングバー）に表示するボタンは、マイクボタン・テキスト入力欄・設定ボタン・ドラッグハンドルのみとする。スクリーンショットボタンおよびファイル添付ボタンは表示しない。

#### Scenario: トップバーにスクリーンショットボタンが表示されない
- **WHEN** アプリが起動してトップバーが表示される
- **THEN** スクリーンショット（カメラ/カーソル）ボタンが存在しない

#### Scenario: トップバーにファイル添付ボタンが表示されない
- **WHEN** アプリが起動してトップバーが表示される
- **THEN** ファイル添付（クリップ）ボタンが存在しない

#### Scenario: 残りのボタンは正常に動作する
- **WHEN** トップバーが表示される
- **THEN** マイクボタン、テキスト入力欄、設定ボタン（⚙️）、ドラッグハンドルがすべて表示され、それぞれ機能する

#### Scenario: 設定ボタンのアイコンがギアマークである
- **WHEN** トップバーが表示される
- **THEN** 設定（ダッシュボード）を開くボタンのアイコンが SettingsIcon（⚙️ ギアマーク）で表示される

#### Scenario: 設定ボタンのツールチップが「設定を開く」である
- **WHEN** ユーザーが設定ボタンにホバーする
- **THEN** ツールチップに「設定を開く」と表示される
